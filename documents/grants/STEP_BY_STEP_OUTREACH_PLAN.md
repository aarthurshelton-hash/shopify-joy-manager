## PHASE 1: REGISTRATIONS (Days 1-3) — Do These FIRST

These are prerequisites that gate everything else. Start all in parallel.

### Step 1: Canadian Government Registrations

- [ ] **Verify CRA Business Number** — Confirm En Pensent has active BN with RC (corporate tax) account
  - URL: https://www.canada.ca/en/revenue-agency/services/tax/businesses/topics/registering-your-business/register.html
  - If not registered: register online (instant for sole prop, 1-2 weeks for corp)

- [ ] **Call NRC IRAP** — Request Industrial Technology Advisor (ITA) assignment
  - Phone: 1-877-994-4727
  - URL: https://nrc.canada.ca/en/support-technology-innovation
  - This is your HIGHEST PROBABILITY Canadian grant ($50K-$1M)
  - Takes 2-4 weeks to get ITA assigned — start TODAY

- [ ] **Register on Innovation Canada Portal**
  - URL: https://innovation.ised-isde.canada.ca/innovation/s/?language=en

- [ ] **Register on IDEaS Portal** (Canadian defence innovation)
  - URL: https://www.canada.ca/en/department-national-defence/programs/defence-ideas.html

### Step 2: US Government Registrations (for DARPA, ARPA-E, IARPA)

- [ ] **Request NCAGE code** (NATO entity code — required for SAM.gov as foreign entity)
  - URL: https://eportal.nspa.nato.int/AC135Public/scage/CageList.aspx
  - Takes 1-2 weeks

- [ ] **After NCAGE arrives:** Register on SAM.gov
  - URL: https://sam.gov/
  - Takes 3-5 weeks for foreign entities

- [ ] **After SAM.gov:** Register on Grants.gov
  - URL: https://www.grants.gov/

### Step 3: Tax Credits

- [ ] **Contact SR&ED advisor** — All current R&D qualifies for 35% refundable tax credit
  - This is FREE MONEY on R&D already done
  - Find advisor via: https://www.canada.ca/en/revenue-agency/services/scientific-research-experimental-development-tax-incentive-program.html
