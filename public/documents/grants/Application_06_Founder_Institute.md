## Company Description

**En Pensent** builds the Universal Grid Portal — a single 64-cell pattern recognition architecture that learns from ANY sequential sensor domain without retraining.

The same grid that beats Stockfish 18 at chess outcome prediction (74.18% vs 71.52%, +2.67pp, n=2,804,090 live games, z>600, p≈0) also achieves nuclear reactor fault detection at binary F1 100% (+11pp vs published Bi-LSTM literature) and chemical fault detection at F1 93.3% (+20.6pp vs industry baseline). All running 24/7 on production infrastructure. All SHA-256 timestamped. All independently verifiable.

## The Problem

Every industrial AI company builds domain-specific models. A nuclear fault detector cannot be retrained for battery degradation. A chess engine cannot analyze reactor safety. Every new application requires a new model, new data pipeline, new training run, new maintenance contract. The total cost to AI-enable industrial safety monitoring across nuclear, chemical, battery, and energy sectors is currently hundreds of billions of dollars — and most of it is redundant.

---

## The Solution

The Universal Grid Portal encodes any sensor stream as a colored temporal pattern on a 64-cell grid. The architecture learns structure from the pattern — not from domain knowledge. A new domain requires a ~100-line adapter to map its sensor channels to grid colors. Same grid. Same inference pipeline. Same self-learning threshold engine.
